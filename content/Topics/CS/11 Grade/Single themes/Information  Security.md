Data Integrity – a requirement for data to be valid, accurate, consistent. There are two main checks to make sure a database is accurate: 
- **verification** checks that the data is actually the data you want. 
- **validation** checks that the correct type of data is entered

**Data Privacy** – a requirement for data to be available to authorized users. 
**Data Security** – a requirement for data to be recoverable if lost or corrupted.
