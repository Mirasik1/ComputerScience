Data backup – additional copy of data that can be used for restore and recovery purposes. Backups are often made on separate storage drives or on a different server.
## Types of Backup
- full backup – copy every item of data, creating a complete second version of what is stored.
- incremental backup – copy the data that has changed since the last full backup is stored
- differential(cumulative) backup – copy only save changes since the last backup of any kind

**Disk mirroring (RAID 1)** – replication of data to two or more disks. Disk mirroring is a good choice for applications that require high performance and high availability.
