## Data flow Diagram 
- a way of representing a flow of data through a process or a system (usually an information system).

## DFD symbols:
- Process - A process represents a transformation or manipulation of data
- Data Flow - A data flow represents the movement of data from one process to another or from a process to a data store
- Data Store - A data store represents a storage location for data
- Source sink / External Entity - An external entity represents a source or destination of data that is external to the system being analyzed.
