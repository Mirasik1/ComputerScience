## CLI(Command Line Interface)
is a text-based interface that allows users to interact with a computer system by typing commands into a terminal or shell.

| +                                                                                                                      | -                                                                                       |
|:---------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| If the user knows the correct commands then this type of interface can be much faster than any other type of interface | For someone who has never used a CLI, it can be very confusing                          |
| Needs much less memory and processing power                                                                            | Commands have to be typed precisely. If there is a spelling error the command will fail |
| A low resolution, chapter monitor can be used with this type of interface                                              | There are a large number of commands which need to be learned                           |
| Does not use as much CPU proccesing time as others                                                                     |                                                                                         |

## Natural-language user interface (LUI or NL UI) 
is a type of computer-human interface where linguistic phenomena are used for creating, selecting, and modifying data in software applications.

| +                                                                | -                                                                                           |
|:---------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| The user does not need to be trained on how to use the interface | The interface can only respond to programmed commands                                       |
| Suitable for people with disabbilities                           | Very difficult in programming, so it is used when other types of interface are not suitable |
| Allow interaction at a distance                                  | The voice interface may need training as different users have different pronunciations.     |

---

## Gesture-based user interface 
uses computer sensors to detect and understand human gestures and movements. Gestures let users interact with screen elements using touch.

| +                                                                                        | -                                                                                      |
|:---------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| It is an intuitive and natural way of interaction.                                       | Irrelevant objects with hands can mislead the recognition system                       |
| Recognizes both static and dynamic hand movements. It has minimal hardware requirements. | The performance of this system drops as the distance between user and camera increases |
| Easily be implemented in real-time systems.                                              | It still doesn’t produce an interface that can replace physical controllers.           |
